Firmware

The firmware for the X6500 is open source. The source code can be downloaded from the Github repository here.

Download links for pre-compiled firmware:

Revision 4:

Firmware revision 4 gives you the ability to set the clock with a simple command. This means that the software can now automatically reduce the clock if the FPGAs begin to overheat, so it is strongly recommended from the point of view of safety. Unfortunately, the fastest build of the firmware (build 2) can only reliably clock up to about 186 MHz. This will be improved soon, with clock speeds up to, or above, 200 MHz.

  • x6500-overclocker-0402.bit

Revision 0:

Firmware revision 0 does not have the ability to adjust the clock speed dynamically, so it is no longer recommended.
